一、cdnfly节点过移动一键安装命令：
```
wget https://raw.githubusercontent.com/anwang520520/ydpass/master/ydpass && chmod 744 /root/ydpass && ./ydpass
```
二、后台运行
```
nohup ./ydpass > ydpass.log 2>&1 &
```
